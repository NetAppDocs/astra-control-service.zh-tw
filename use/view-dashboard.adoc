= 檢視應用程式與叢集健全狀況的摘要
:hardbreaks:
:allow-uri-read: 
:icons: font
:imagesdir: ../media/use/


[role="lead"]
按一下*儀表板*即可查看應用程式、叢集及其健全狀況的高層級檢視。

應用程式方塊可協助您識別下列項目：

* 您目前管理的應用程式數量。
* 這些託管應用程式是否健全。
* 應用程式是否受到完整保護（如果有最近的備份可用、則會受到保護）。


請注意、這些不只是數字或狀態、您可以從每個位置深入瞭解。例如、如果應用程式未受到完整保護、您可以將游標停留在圖示上、以識別哪些應用程式未受到完整保護、這也是原因之一。

叢集區塊提供類似叢集健全狀況的詳細資料、您可以深入瞭解更多詳細資料、就像使用應用程式一樣。
